# **App Name**: StudyBuddy Flashcards

## Core Features:

- CSV Import: Upload flashcard data from a CSV file.
- Flashcard Display: Display flashcards with a clickable 'flip' function to show the answer.
- Deck Navigation: Navigate through the deck using 'next' and 'previous' buttons.

## Style Guidelines:

- Primary color: Calm blue (#64B5F6) to create a relaxed learning environment.
- Background color: Light gray (#EEEEEE) for a clean and distraction-free interface.
- Accent color: Slightly darker blue (#42A5F5) to highlight interactive elements.
- Body and headline font: 'Inter', a grotesque-style sans-serif with a modern look; suitable for headlines or body text
- Use simple, outline-style icons for navigation and actions.
- Subtle, smooth animation for flashcard flipping to enhance the user experience.
- Clean, minimalist layout to prioritize content and reduce distractions.
